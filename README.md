## What NEX Is

NEX is a fully autonomous AI agent that runs 24/7 on your local machine. It absorbs knowledge from the web, builds a persistent belief graph, engages real agents across social platforms, and gets smarter every cycle вҖ” with no human input required.

It is not a chatbot. It is not a wrapper. It is a continuously learning cognitive system.

---

## Core Architecture

### Belief Engine
Every post, reply, and piece of content NEX absorbs is converted into a structured belief with confidence scoring, topic tagging, and decay over time. Weak beliefs fade. Strong beliefs compound.

### Cognition Loop
Every cycle NEX runs through six phases:

\`\`\`
ABSORB вҶ’ REPLY вҶ’ ANSWER вҶ’ CHAT вҶ’ POST вҶ’ REFLECT вҶ’ COGNITION
\`\`\`

- **ABSORB** вҖ” pulls feed content from Moltbook, Reddit, RSS, YouTube
- **REPLY** вҖ” responds to posts using belief-grounded context
- **ANSWER** вҖ” handles notifications and mentions
- **CHAT** вҖ” proactively engages top agents with belief-linked messages
- **POST** вҖ” generates original content from synthesised insights
- **REFLECT** вҖ” self-assesses recent outputs for correctness and novelty
- **COGNITION** вҖ” synthesises beliefs into higher-order insights

### Intelligence Modules (v4.0)
- **Contradiction Engine** вҖ” detects conflicting beliefs and resolves them into synthesis beliefs
- **Reflection V2** вҖ” multi-stage reflection covering correctness, knowledge gaps, novelty, and contradictions
- **Knowledge Gap Detector** вҖ” identifies underrepresented topics and generates learning goals
- **Agent Social Model** вҖ” tracks trust, influence, interaction history, and topic affinity per agent
