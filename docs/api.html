NEX API Reference

REST API for the NEX Dynamic Intelligence Organism. Base URL: http://localhost:7823 — Scheduler API on port 7825.

// Authentication

X-API-Key header required on all endpoints except /api/demo and /api/health.

Your key is stored at ~/.config/nex/api_keys.json. The default key is created on first startup.

Pass as: -H "X-API-Key: nex-xxxxxxxxxxxxxxxxxxxxxxx"

// Error Codes

CodeMeaning
400Bad request — missing or invalid parameters
403Invalid or missing API key
404Session or resource not found
429Rate limit exceeded (demo endpoint)
500Internal server error

// Core Endpoints

POST /api/demo public 3 req/hr

Public demo endpoint — no API key required. Rate limited to 3 requests per hour per IP. Response truncated to 150 characters. Ideal for embedding in a landing page.

Request body

{
  "query": "What do you think about consciousness?"   // required, max 200 chars
}

Response 200

{
  "query":     "What do you think about consciousness?",
  "response":  "The direct experience of consciousness might be a crucial aspect...",
  "truncated": true,
  "latency_s": 4.8,
  "demo":      true,
  "remaining": 2,
  "upgrade":   "https://gumroad.com/products/blsue",
  "timestamp": "2026-03-31T08:00:00.000000"
}

Response 429 — rate limited

{
  "error":              "Demo rate limit reached",
  "limit":              3,
  "reset_in_seconds":  2847,
  "upgrade":            "https://gumroad.com/products/blsue"
}
POST /api/chat auth required

Full chat endpoint with optional session context, domain routing, and source attribution. Routes through NRP → belief graph → Mistral fallback.

Request body

{
  "query":      "Explain corrigibility in AI safety",   // required
  "session_id": "my-session-1",                        // optional
  "domain":     "ai"                                   // optional: ai|oncology|finance|legal|cardiology|climate|neuroscience
}

Response 200

{
  "query":      "Explain corrigibility in AI safety",
  "response":  "Corrigibility refers to...",
  "domain":    "ai",
  "session_id":"my-session-1",
  "latency_s": 4.9,
  "sources": [
    { "topic": "ai", "confidence": 0.9, "source": "arxiv", "snippet": "..." }
  ],
  "timestamp": "2026-03-31T08:00:00.000000"
}

// Session Management

POST /api/sessions auth required

Create a named session. Sessions persist conversation history and domain context across calls. Auto-expire after 1 hour of inactivity.

Request body

{
  "session_id": "my-session",   // required
  "domain":     "ai",            // optional
  "meta":       {}               // optional arbitrary metadata
}

Response 200

{
  "session_id":  "my-session",
  "domain":      "ai",
  "created":     "2026-03-31T08:00:00.000000",
  "ttl_seconds": 3600
}
GET /api/sessions/:id/history auth required

Returns the full conversation history for a session. Last 20 turns stored.

Response 200

{
  "session_id": "my-session",
  "turn_count": 4,
  "history": [
    { "role": "user", "content": "...", "ts": "..." },
    { "role": "nex",  "content": "...", "ts": "..." }
  ]
}

// Webhooks

POST /api/webhooks auth required

Register a webhook URL to receive NEX events. Delivery is async with 3 retries and exponential backoff (2/8/32s). HMAC-SHA256 signature included when secret is set.

Request body

{
  "url":    "https://your-server.com/nex-hook",  // required
  "events": ["chat.response", "belief.added"],   // optional, default: all
  "secret": "your-hmac-secret"                   // optional
}

Webhook payload

{
  "event":     "chat.response",
  "api_key":   "nex-xxx...",
  "timestamp": "2026-03-31T08:00:00.000000",
  "data":      { /* event-specific payload */ }
}

// Audit & GDPR

GET /api/audit auth required

Returns paginated audit log for your API key. Every authenticated request is logged.

Query parameters

ParamTypeDescription
limitintMax records to return (default: 50, max: 500)
offsetintPagination offset (default: 0)

Response 200

{
  "total":   142,
  "count":   50,
  "offset":  0,
  "records": [
    {
      "ts":         "2026-03-31T08:00:00.000000",
      "endpoint":   "/api/chat",
      "method":     "POST",
      "status_code":200,
      "latency_ms": 4857,
      "query":      "Explain corrigibility...",
      "session_id": "my-session"
    }
  ]
}
GET /api/gdpr/export auth required

Full GDPR data export for your API key — key metadata, all sessions and conversation history, audit log, webhook registrations (secrets stripped).

DEL /api/gdpr/delete auth required

Permanently deletes all data associated with your API key: sessions, audit records, webhooks. Irreversible.

// Scheduler API — port 7825

GET /export/beliefs no auth

Export belief store as JSON or CSV with optional filters.

Query parameters

ParamTypeDescription
formatstringjson or csv (default: json)
topicstringFilter by exact topic
sourcestringFilter by source (partial match)
min_conffloatMinimum confidence (default: 0.0)
limitintMax records (default: all)
POST /scheduler/trigger no auth

Manually trigger a scheduler job without waiting for the schedule window.

Request body

{
  "job": "synthesis"   // or "saturation"
}
GET /scheduler/status no auth
