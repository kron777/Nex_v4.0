In __init__, accept and store the engine:

    def __init__(self, ..., social_engine=None):
        ...
        self.social_engine = social_engine
        self.conversation_history = []   # add if not already present

In your REPLY phase, before the LLM call in express():

    # ── Social context (belief-graph-native) ──────────────────
    if self.social_engine:
        from nex_social_engine import social_system_prompt
        ctx = self.social_engine.analyse(
            message=user_message,
            conversation_history=self.conversation_history
        )
        system_prompt = social_system_prompt(ctx, base_prompt=self.base_system_prompt)
    else:
        system_prompt = self.base_system_prompt

    # Pass system_prompt to your /completion call as usual
    # ──────────────────────────────────────────────────────────

After getting the reply, append to history:

    self.conversation_history.append(user_message)
    self.conversation_history.append(nex_reply)
    # keep last 16 turns (8 exchanges)
    self.conversation_history = self.conversation_history[-16:]

(Engineering rule #11/#12 from session state)
The system_prompt should be injected as part of the
`prompt` field, NOT as a messages[] array.

Typical format for /completion:

    payload = {
        "prompt": f"{system_prompt}\n\nUser: {user_message}\nNEX:",
        "n_predict": 512,
        "stop": ["User:", "\n\n"]
    }

If you're using /v1/chat/completions format instead,
system_prompt goes in messages[0] as role="system".

WHAT THIS BUYS NEX (social skills delta)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Before: LLM picks tone and depth on its own.
        Social behaviour is the LLM's personality, not NEX's.

After:  Every social reply is shaped by:
        - Which of NEX's beliefs the message activated
        - How certain she is on this topic (temperature)
        - What position she actually holds (stance)
        - What the person has revealed about their own view (overlap)
        - What she needs right now (drive urgency)
        - Whether she holds a genuine tension here (contradiction)

        The LLM only translates. NEX decides.

This is the Master Map principle applied to social:
  "NEX is the author. The LLM is the pen."
